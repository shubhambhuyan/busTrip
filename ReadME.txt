This project is created using HTML, CSS and JAVASCRIPT.
and few opensource technologies like, bootstrap and google api (for static map)
To run this project u just have click on index.html
and start exploring.

for experiment purpose only 4 bus numbers have been added.
the busses which are in data base are listed out in data.json
where id denotes the bus number(route number)
source and destination determine the starting and end stops.
and stops is an array denoting the intermediate bus stops.
